# Videos locales (Kitch)

Coloca aquí los videos optimizados usados por la web.

## Hero video (requerido para la experiencia completa)

- `restaurant-operations-hero.mp4` (principal)
- `restaurant-operations-hero.webm` (opcional)

El Hero intentará cargar el archivo en:

- `/videos/restaurant-operations-hero.mp4`

Si el video no existe o no carga, **la UI no se rompe**: se usa un fallback con gradientes oscuros.

## Recomendación de exportación

- Formato: **MP4 H.264** (y WebM opcional)
- Resolución: **1920×1080** o **1600×900**
- Duración: **8–15 s**
- Loop: **suave**, sin cortes notorios
- Audio: **sin audio**
- Peso ideal: **< 5 MB**
